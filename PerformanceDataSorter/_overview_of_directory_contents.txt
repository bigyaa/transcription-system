This directory contains files for supporting the processing of data related to the transcription and diarization
of recordings from the International Storytelling Center.  As of 27 June 2024, it contains the following files and directories:

-.  _overview_of_directory_contents.txt - this file
-.  collapsing peformance data into one xml document.txt
    -.  a document that describes how to use Windows ports of standard Unix utilities to generate this 
        file of performance data
-.  sort transcription performance data.xsl - an application that sorts an XML file that contains performance data
    generated by the AudioDispatcher, using optional user-specified primary and secondary sort keys
-.  sample output, with input - examples of output generated by sort transcription performance data.xsl
